PID制御関数ソースの使い方	
作成日2016/2/10
最終更新日2016/3/12

　このC言語ソースには複数のタイプに分けている。

	・第1世代：ver1.xx
	・第2世代:ver2.xx

	第1世代について(Ver1.00開発中)
	　第1世代はC言語の初心者向けに作成し、シンプルにたった1つの関数に集約している。
	使い方はソース内の関数を丸ごとコピペして使うかソースファイル、ヘッダファイルを共にビルドして
	使用する。関数の説明はヘッダファイルに、パラメータ調整はソースファイルに記述してある。
	　機能については主に通常の偏差PID、微分先行型P-ID、比例微分先行型I-PDアルゴリズムを実装している。
	　現在単精度浮動小数点型のみ対応するものを開発中。

	第2世代について（最新Ver2.20）
	　第2世代はC言語の～中級者向けに作成している。ヘッダファイルの見方が分かること、構造体、ポインタの知識があれば充分である。
	　ソースファイル、ヘッダファイルを共に使用することを前提とし、部分的なコピペは動作は保証しない。第1世代では関数の引数が明らかに多すぎるのでパラメータ設定用とPID演算用にインターフェースを分散している。また、複数のPID関数を利用することを想定してデータとロジックを分離する仕様としている。
	　使い方はヘッダファイルを参照すること。またPIDパラメータの調整法はソースファイルに記述してある。
	 機能については主に通常の偏差PID、微分先行型P-ID、比例微分先行型IPDアルゴリズムを実装している。
	 現在単精度浮動小数点型のみ対応している。
